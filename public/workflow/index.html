How the application works — interactive workflow

🔐

Login / Register

Students & admins sign in. Auth state managed via React Context.

  • Page: /login
  • Components: AuthContext, UserContext
  • Roles: USER · ADMIN
🏠

Dashboard

Central hub — widgets, feed, quick-access cards, and sidebar navigation.

  • Page: / (page.tsx — 17 kB)
  • Components: Sidebar, FeedList, PredictorWidget, StudyBotWidget
  • Theme: ThemeContext (dark/light)
📦

Resource Vault

Upload PDFs & photos, OCR extraction via Tesseract.js, LLM summaries.

  • Page: /vault
  • API: /api/extract, /api/notes, /api/summarize
  • Components: UploadCard, MyNotes
  • OCR: Tesseract.js 7
📊

Attendance

Track attendance per subject, predict skip-safe probability.

  • API: /api/attendance
  • Components: EnhancedPredictor, PredictorWidget
  • Model: AttendanceLog
🎓

CGPA Calculator

Enter grades & credits, compute cumulative GPA instantly.

  • API: /api/cgpa
  • Components: CGPANavigator
🤖

AI Study Bot

Chat with an AI assistant — ask questions, get study tips & summaries.

  • Page: /assistant
  • API: /api/chat, /api/summarize
  • Components: StudyBotWidget
📈

Analytics

Visual dashboards of your academic performance & trends.

  • Page: /analytics
  • Data: Aggregated from Attendance & CGPA
⚙️

Next.js API Routes

Server-side route handlers under /api/* — REST endpoints for every feature.

  • /api/attendance
  • /api/cgpa
  • /api/chat
  • /api/extract
  • /api/notes
  • /api/summarize
🗄️

PostgreSQL + Prisma

Three core models — User, Resource, AttendanceLog. Managed via Prisma ORM.

  • User: id, name, university, year, department, role
  • Resource: uploaderId → User, fileType, summaryText
  • AttendanceLog: userId → User, subject, status, skipSafeProbability
